
# TEKSI Module Management Tool (TMMT)

This tool eases the installation (and in future also) the upgrade of any TEKSI module.

The current version allows to install:
 * [TEKSI Wastewater & GEP](https://github.com/teksi/wastewater/)

![TEKSI Module Management Tool (TMMT)](docs/en/images/tmmt_tww_2025_0_2_success.jpg?raw=true)

Here you can get an impression how it works: [TMMT is LIVE for TEKSI Modules installation](https://github.com/teksi/TMMT/discussions/34)


The generic module is developped here: https://github.com/opengisch/oqtopus
